Vertical Pull Progressions

  • Assisted Pull-Ups: 3 sets x 8-10 reps
  • Negative Pull-Ups: 3 sets x 5 reps
  • Pull-Ups: 3 sets x 5-10 reps
  • Chin-Ups: 3 sets x 5-10 reps
  • Muscle-Up Progressions: 3 sets x 3-5 reps

Horizontal Pull Progressions

  • Inverted Rows: 3 sets x 10-15 reps
  • Ring Rows: 3 sets x 8-12 reps
  • One-Arm Rows: 3 sets x 6-8 reps
  • Australian Pull-Ups: 3 sets x 12-15 reps

Vertical Push Progressions

  • Incline Push-Ups: 3 sets x 10-15 reps
  • Pike Push-Ups: 3 sets x 5-10 reps
  • Handstand Push-Ups (against wall): 3 sets x 5-8 reps
  • Handstand Push-Ups (freestanding): 3 sets x 3-5 reps

Horizontal Push Progressions

  • Push-Ups: 3 sets x 10-15 reps
  • Diamond Push-Ups: 3 sets x 8-12 reps
  • Archer Push-Ups: 3 sets x 8-12 reps
  • Decline Push-Ups: 3 sets x 10-12 reps

Core Progressions

  • Plank: 3 sets x 30-60 seconds
  • Leg Raises: 3 sets x 10 reps
  • Dragon Flags: 3 sets x 3-5 reps

  • Mountain Climbers: 3 sets x 30 seconds

Statics Progressions

  • Front Lever Hold Progressions: 3 sets x 5-10 seconds

Program 1: Beginner Strength and Mobility

This program is designed for those just starting out with bodyweight training.

  • Push-Ups: 3 sets x 10-12 reps
  • Bodyweight Squats: 3 sets x 12-15 reps
  • Plank: 3 sets x 30-45 seconds
  • Inverted Rows: 3 sets x 8-10 reps
  • Leg Raises: 3 sets x 10 reps

Program 2: Intermediate Strength and Hypertrophy

This program is for those who are ready to progress to more advanced exercises.

  • Incline Push-Ups: 3 sets x 10-15 reps
  • Bulgarian Split Squats: 3 sets x 12-15 reps
  • Dragon Flags (Progressions): 3 sets x 5-8 reps
  • Ring Rows: 3 sets x 8-12 reps
  • Hollow Body Hold: 3 sets x 20-30 seconds

Program 3: Advanced Strength and Power

This program is for those who are looking to really push their limits and increase strength.

  • Handstand Push-Ups: 3 sets x 5-8 reps
  • Pistol Squats: 3 sets x 5-8 reps per leg
  • Muscle-Ups: 3 sets x 3-5 reps
  • Front Lever Progressions: 3 sets x 5-8 seconds hold
  • Archer Push-Ups: 3 sets x 8-12 reps

Program 4: Bodyweight Endurance & Stamina

This program is for those who want to focus on endurance, conditioning, and stamina.

  • Push-Ups (Max Reps in 1 Minute)
  • Bodyweight Squats (Max Reps in 1 Minute)
  • Burpees: 3 sets x 15-20 reps
  • Mountain Climbers: 3 sets x 30 seconds
  • Plank: 3 sets x 60 seconds

Program 5: Full Body Strength & Mobility

This program integrates strength training with mobility work, ensuring balanced development.

  • Push-Ups: 3 sets x 8-12 reps
  • Squat to Lunge Transitions: 3 sets x 12 reps
  • Plank to Push-Up: 3 sets x 10-12 reps
  • Hip Flexor Stretch: 3 sets x 30 seconds per side
  • Shoulder Mobility Drills: 3 sets x 10 reps
