Strengthmind Bodyweight Mastery

Vertical Pull Progressions

  • Assisted Pull-Ups: 3 sets x 8-10 reps
  • Negative Pull-Ups: 3 sets x 5 reps
  • Pull-Ups: 3 sets x 5-10 reps
  • Chin-Ups: 3 sets x 5-10 reps
  • Muscle-Up Progressions: 3 sets x 3-5 reps

Horizontal Pull Progressions

  • Inverted Rows: 3 sets x 10-15 reps
  • Ring Rows: 3 sets x 8-12 reps
  • One-Arm Rows: 3 sets x 6-8 reps
  • Australian Pull-Ups: 3 sets x 12-15 reps

Vertical Push Progressions

  • Incline Push-Ups: 3 sets x 10-15 reps
  • Push-Ups: 3 sets x 8-12 reps
  • Handstand Push-Ups (against a wall): 3 sets x 3-5 reps
  • Archer Push-Ups: 3 sets x 5-8 reps

Horizontal Push Progressions

  • Incline Push-Ups: 3 sets x 10-15 reps
  • Push-Ups: 3 sets x 8-12 reps
  • Decline Push-Ups: 3 sets x 6-10 reps
  • Clap Push-Ups: 3 sets x 5-8 reps

Core Progressions

  • Plank: 3 sets x 30-60 seconds
  • Side Plank: 3 sets x 20-30 seconds each side
  • Hanging Leg Raises: 3 sets x 5-10 reps
  • Dragon Flags: 3 sets x 3-5 reps

Statics Progressions

  • Wall Sit: 3 sets x 30-60 seconds
  • Planche Lean: 3 sets x 10-20 seconds
  • Front Lever Hold: 3 sets x 5-10 seconds
  • Back Lever Hold: 3 sets x 5-10 seconds

Gear Needed

For the workouts, you will need:

  • Pull-Up Bar
  • Rings (optional)
  • Yoga Mat
  • Resistance Bands (optional)

Programs of Varying Difficulty

Beginner Program

  • Assisted Pull-Ups: 3 sets x 8-10 reps
  • Incline Push-Ups: 3 sets x 10-15 reps
  • Plank: 3 sets x 30 seconds

Intermediate Program

  • Pull-Ups: 3 sets x 5-10 reps
  • Push-Ups: 3 sets x 8-12 reps
  • Hanging Leg Raises: 3 sets x 5-10 reps

Advanced Program

  • Muscle-Up Progressions: 3 sets x 3-5 reps
  • Handstand Push-Ups: 3 sets x 3-5 reps
  • Dragon Flags: 3 sets x 3-5 reps
